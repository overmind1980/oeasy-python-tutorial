### 进步

- 输入法在桌面
	- 点击输入法之后
	- 右键右下角小图标

```
dayup = 1
dayfactor = 0.01
for day in range(365):
	dayup *= (1 + dayfactor)
print("进步幅度{:.3f}".format(dayup))
```

- 这里的{:.3f}指的是保留三位有效数字

- 别小看这个例子哦
- 人工智能也是
	- 由各种初始值
	- 各种因子等等
	- 一步步具有的智能
- 有进步就有退步

### 退步

- 同样可以得到退步幅度
- 每天退步一点点

```
dayup = 1

print("退步幅度{:.3f}".format(dayup))
```

### 药方输出

```
s_fangzi = "麻黄9g 炙甘草6g 杏仁9g 生石膏15-30g(先煎) 桂枝9g 泽泻9g 猪苓9g 白术9g 茯苓15g 柴胡16g 黄芩6g 姜半夏9g 生姜9g 紫菀9g 冬花9g 射干9g 细辛6g 山药12g 枳实6g 陈皮6g 藿香9g"
l_drugs = s_fangzi.split(" ")
i_num = 0
for s_drug in l_drugs:
    i_screen_len = 0
    for c in s_drug:
        if 0x4e00 <= ord(c) <= 0x9fff:
            i_screen_len += 2
        else:
            i_screen_len += 1
    if i_screen_len >= 8:
        s_drug = s_drug + "\t"
    elif i_screen_len >= 4:
        s_drug = s_drug + "\t\t"
    print(s_drug,end='')
    if i_num % 4 == 3:
        print()
    i_num = i_num + 1
```

#### evaluateequ.py

- 这个程序计算数列 
	- *1/x+1/(x+1)+1/(x+2)+ ... +1/n*
	- 设 x = 1，n = 10

```python
#!/usr/bin/env python3
sum = 0
for i in range(1, 11):
    sum += 1.0 / i
    print("{:2d} {:6.4f}".format(i , sum))
```
