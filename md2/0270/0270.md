	- `ps -elf` 查看 所有进程的 信息
- 杀死 进程
	- `kill -9 PID` 给进程发送死亡信号
- 后台 运行
	- `nohup python3 show.py>>s.log &`

```python
#!/usr/bin/python
i = 0
import time
while True:
    i = i + 1
    print(i, "===", time.asctime())
    time.sleep(1)
```

### 观察 父子关系

- 可以查看到
	- 这进程之间有父子关系
	- `pid` 是指进程的 `id`
	  - **p**rocess **id**
	- `ppid` 是指 **父**进程 **id**
	  - **p**arent **p**rocess **id**
