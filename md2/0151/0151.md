# 比较运算符

## 回忆

- 单个的 if 
	- 是单分支结构
- if - else 
	- 是双分支结构
- else 和 if 配对的
	- 满足 if条件 
		- 就运行 if
	- 不满足 if条件
		- 就运行 else

### 转义问题

- 解决\的问题
  - `:4,$s/\\/\\\\/g`
	 - `:` 执行命令行模式
	 - `%` 对所有行执行命令
	 - `s` 执行的是替换命令
	 - `\\` 代表一个反斜杠
	 - `\\\\` 换成两个反斜杠
	 - `g` 代表global 每一行中所有的匹配项都替换
