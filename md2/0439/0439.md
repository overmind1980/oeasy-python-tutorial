### 二进制的作用

- 这样流头就可以自由地`相对`运动了

### mode

- mode 模式设置为 rb
  - r - read - 读
  - b - binary 
- 返回值变成了字节流
  - b"222\n"

![图片描述](uid1190679-20210823-1629704619215.png)

- 二进制 和 纯文本(text)相对



- whence 
	- 除了相对于头0、
	- 相对于当前1
	- 还有一种相对于尾的模式2
- 怎么用呢？

#### seek

- whence = 2 是从尾部计算
	- -3 就是倒数第 3 个字符

![图片描述](uid1190679-20210823-1629704866829.png)

### 解码

