### 调整

- 调整为 % 显示方法

```
for i in range(1, 10):
    for j in range(1, 10):
        print("%d" % i, "*", "%d" % j, "=", \
              "%2d" % (i * j), \
              sep="", end=" ")
    print()
```

- 显示结果

![图片描述](uid1190679-20230217-1676592876595.png)

- 这里对齐的核心是
	 - "%2d" % (i * j)
	 - 要让 (i * j) 以 decimal 的方式显示
	 - 并且 占`两`位
