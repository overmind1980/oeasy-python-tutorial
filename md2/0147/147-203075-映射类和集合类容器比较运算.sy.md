### 大于小于

```

{1, 2} > {1, 2, 3}
```

### 比较的对称性

- 对称性
	- 等号 两边 可以 对换
	- 不等号 两边 可以 对换
	- 大于号 等价于  两边对换后 换小于号

### 比较的传递性

```python

```

- 能推出

```python

```

- imply 就是暗示的意思
	- 可以判断出来

### 应用

```
base_auth = {"拍照", "存储"}
full_auth = {"拍照", "存储", "定位", "麦克风"}

print(base_auth < full_auth)   # True
print(full_auth > base_auth)   # True
```

### 总结

- 这次研究了比较
  - 映射类 map
    - dict 字典
  - 集合类
    - set
    - frozenset

| 比较运算符 | 字典 (dict) | 集合 (set) |
|------------|-------------|------------|
| `==` | 键值对**完全相同**则为 True<br>顺序不影响 | 元素**完全相同**则为 True<br>顺序不影响 |
| `!=` | 键/值有任一不同 → True | 元素有任一不同 → True |
| `<、<=、>、>=` | ❌ **直接报错**<br>字典不支持大小比较 | ✅ **支持**<br>表示**子集/超集**关系 |

- 比较运算符还有
  - 对称性
  - 传递性
  - 否定之否定原理
- COMPARISION 中还有个 
	- 成员测试
	- Membership test
