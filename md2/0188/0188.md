### 代码

```
while True:
    numbers = []
    while True:
        num = input("请输入商品价格（输入end停止输入）：")
        if num == "end":
            break
        try:
            num = float(num)
            numbers.append(num)
        except ValueError:
            print("输入不合法，请输入有效的浮点数或end。")
    total = sum(numbers)
    print(f"本次输入的数字总和为：{total}")
```
