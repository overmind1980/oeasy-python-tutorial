### 提前声明

- 需要先声明level

```
level = "low"
t = float(input("Your temperature:"))
if (t > 38):
    level = "high"
print("Temperature is " + level)
```

- 不论 if 条件是否满足
	- level 都最好提前声明了
	- 而不是 在某个分支里 被声明
- 这就避免 了
	- level 没声明 就使用
