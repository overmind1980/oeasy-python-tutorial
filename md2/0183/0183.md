### reversed 方法

- 我们要的是排好逆序

  - 第三部转化为列表
- 这些都不会影响列表 l 本身的排序
- 我们来看看帮助

### reversed 细节

![图片描述](uid1190679-20211007-1633588278293.png)

- reversed 是一个类
- 返回一个可迭代的对象
- 这个可迭代的对象可以被转化为列表
- 我们可以再分一下象限

### 比较

| 表               | sort         | reverse      | sorted   | reversed       |
| ---------------- | ------------ | ------------ | -------- | -------------- |
| 所属             | 列表成员函数 | 列表成员函数 | 内建函数 | 内建的类       |
| 是否会影响原列表 | 会           | 会           | 不会     | 不会           |
| 返回类型         | None         | None         | 新列表   | 新可迭代类对象 |
| 排序方式         | 正序         | 逆序         | 正序     | 逆序           |

- 正序和逆序都属于排序
- 其实最好用的还是 sorted
- 因为 sorted 可以有一个 reverse 的参数
- 此外还有个乱序
