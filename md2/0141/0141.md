```
t = input("Your temperature:")
if t > 38:
    print("You have a fever!")
```

- TypeError
	- 类型错误
	- 输入得到的类型 是 str
	- 比较大小 两边类型 应该一致

### 改错

- 将输入 转化为 浮点型(float)

```
t = float(input("Your temperature:"))
if t > 38:
    print("You have a fever!")
```

- 逻辑上 没有问题 了
