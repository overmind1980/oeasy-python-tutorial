### 代码实现

```
import bpy

bpy.ops.object.select_all(action="SELECT")
bpy.ops.object.delete()
bpy.ops.mesh.primitive_cube_add(size = 1)
bpy.context.object.scale[0] = 3
bpy.context.object.rotation_euler[1] = 1.5708
```

- 原来
	- 旋转就是
	- 改变object.rotation_euler

![图片描述](../0681/uid1190679-20240523-1716458846820.png)

- <span style="color:red">x</span><span style="color:green">y</span><span style="color:blue">z</span> 三个轴向的旋转值构成一个列表
	- rotation_euler[<span style="color:red">0</span>] 是 <span style="color:red">x</span>轴 保持
	- rotation_euler[<span style="color:green">1</span>] 是 <span style="color:green">y</span>轴 设置为 1.5708
	- rotation_euler[<span style="color:blue">2</span>] 是 <span style="color:blue">z</span>轴 保持

### 角度 弧度

- 弧度制 
	- 将和半径相等的弧长对应的角度
	- 定义为1

![图片描述](../0681/uid1190679-20240302-1709341131723.png)

- 一周 
	- 弧度制 2*π ≈ 6.28
	- 角度制 360°
- 直角
	- 角度制 360 / 4 = 90°
	- 弧度制 2π /4 = π/2 ≈ 1.5708
- 想要在此竖的基础上
	- 再来一个横

### 批量旋转

```
import bpy
from math import pi,cos,sin
r = 3
n = 24
for i in range(n):
    theta = i * pi * 2/n
    pos = (r*cos(theta), r*sin(theta), 0)
    bpy.ops.mesh.primitive_cube_add(size=0.6,location=pos,
    rotation=(0,0,theta)) 
```
