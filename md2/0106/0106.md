- 字符串可以通过del删除切片吗？

### 字符串 删除切片


```
s = "oeasy"
s[1:3]
del s[1:3]
```

- 字符串 支持切片
	- 但 不支持切片删除

![图片描述](uid1190679-20251107-1762497886913.png) 

- 怎么办呢？

### 绕道列表

- 字符串 不支持 del关键字
	- 但是 `列表`支持
	- 先把 字符串 `转化`为 `列表`

```
s = "oeasy"
clist = list(s)
del clist[1:3]
clist
s = ""
for c in clist:
	s = s + c

print(s)
```

- 最后 再把 列表 `转化回` 字符串
	- 直着 过不去
	- 绕过去
