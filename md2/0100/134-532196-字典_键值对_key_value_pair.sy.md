### 字典(dict)的结构

- 数据类型 是 python的基础
	- int、str 是 最基本的 基础数据类型
	- list、tuple、dict 是最基本的容器类型

```
d = {"one": 1, "two": 2}
d.keys()
list(d.keys())
d.values()
list(d.values())
d.items()
list(d.values())
```

- 本质上来说
  - keys 是key的集合
  - values 是value的列表
  - items 是key-value pair的集合
	- 名值对 的 集合
