### COMPARISON

| 运算符 | 含义 |
| ---- | ---- |
| > | 大于 |
| < | 小于 |
| >= | 大于等于 |
| <= | 小于等于 |
| == | 相等（判断值是否相同） |
| != | 不相等（判断值是否不同） |
| is | 身份相同（判断两个对象是同一个） |
| is not | 身份不同（判断两个对象不是同一个） |
| in | 成员存在（判断元素在容器中） |
| not in | 成员不存在（判断元素不在容器中） |
