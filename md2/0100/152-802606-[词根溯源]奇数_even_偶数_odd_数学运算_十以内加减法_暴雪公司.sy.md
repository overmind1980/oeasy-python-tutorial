### 代码

- 上来之后 
	- while True
	- 死循环

```
import random

while True:
    # 生成随机数
    n1 = random.randint(0, 10)
    n2 = random.randint(0, 10)
    correct = n1 + n2
    
    # 打印题目
    print(f"{n1} + {n2} = ", end="")
    
    # 处理用户输入（防非数字报错）
    try:
        result = int(input())
    except ValueError:
        print("\033[38;5;9m ✗ 请输入有效数字！\033[0m")
        continue
    
    # 判断结果
    if result == correct:
        print("\033[38;5;10m ✓ 正确！\033[0m")
    else:
        print(f"\033[38;5;9m ✗ 错误！正确答案是：{correct}\033[0m")
    
    # 选择是否继续
    choice = input("\n是否继续练习？(y/n): ").strip().lower()
    if choice != "y":
        print("练习结束！")
        break

```
