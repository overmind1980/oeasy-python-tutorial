### 序列化

```
import pickle
d_colors = {
    'dahong': {'r': 255, 'g': 33, 'b': 33},

}

```

### 解压

![图片描述](uid1190679-20210924-1632489649972.png)

```
import pickle
d_colors = {
    'dahong': {'r': 255, 'g': 33, 'b': 33},
   'shiqing': {'r': 23, 'g': 34, 'b': 52}
}
b_colors = pickle.dumps(d_colors)
print(d_colors)
print(b_colors)
d2 = pickle.loads(b_colors)
print(d2, type(d2))
```
