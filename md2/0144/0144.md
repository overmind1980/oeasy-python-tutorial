### 容器

```

bool(())
bool({})
bool(set())
```

- **空容器一律为 False**

| 表达式       | 结果  | 说明           |
|-------------|-------|----------------|
| `bool([])`   | False | 空列表         |
| `bool(())`   | False | 空元组         |
| `bool({})`   | False | 空字典         |
| `bool(set())`| False | 空集合         |
