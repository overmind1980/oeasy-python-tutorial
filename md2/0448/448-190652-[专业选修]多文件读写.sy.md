### 整理环境

- 使用vim同时打开 3个文件
	- o0z.data
	- o1z.data
	- o2z.data

```
vi -O o{0..2}z.data
```

- vi 编辑器
  - -O 纵向拆分 window
  - o{0..2}z 对应三个文件
    - o0z.data
    - o1z.data
    - o2z.data
- 这3个文件
	- 扩展名 为 data 
	- 编码格式 为 文本

### 打开多文件

```
i_sum = 0
for i in range(0,3):
    file_name = "o" + str(i) + "z.data"
    f = open(file_name)
    s_num = f.read()
    i_num = int(s_num)
    i_sum += i_num
    f.close()
print(i_sum)
```

- 保存并运行
