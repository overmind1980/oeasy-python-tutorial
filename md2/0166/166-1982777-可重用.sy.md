- 上次学习了 and

	- 都是 True 的时候
		- 才返回 True 
		- 其余都返回 False
- 这个运算符变量可以
	- 有数字、字符串、容器之类的

- 一旦出现 False
	- 就返回逻辑值 False 对应的变量
