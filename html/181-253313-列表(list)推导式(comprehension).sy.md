- 让人写起来很快、而且可读性很强

- 但是一旦实现了语法糖的功能
- 语法糖会让编程工作更快速

  - 看起代码来反而会吃力
- 我的建议是
  - 首先能看懂
  - 至于写不写语法糖的代码
  - 我们不去追求
  - 把基本功练好最重要

### 不加糖的IloveU

```
for y in range(15, -15, -1):#表示从15- -15每隔一个单位取一次值，
    ls=[]
    for x in range(-30, 30):
        tp=((x*0.05)**2+(y*0.1)**2-1)**3-(x*0.05)**2*(y*0.1)**3
        if tp<=0:  ##点在图像内部
            a=''.join('IloveU'[(x - y) % 6]) #加字母
        else:   ##点在图像外部
            a=''.join(' ')
        ls.append(a)
    s=''.join(ls)
    print(s)
```

- 更通用的格式

```
text = "IloveU"
for y in range(15, -15, -1):#表示从15- -15每隔一个单位取一次值，
    ls=[]
    for x in range(-30, 30):
        tp=((x*0.05)**2+(y*0.1)**2-1)**3-(x*0.05)**2*(y*0.1)**3
        if tp<=0:  ##点在图像内部
            a=''.join(text[(x - y) % len(text)]) #加字母
        else:   ##点在图像外部
            a=''.join(' ')
        ls.append(a)
    s=''.join(ls)
    print(s)
```

- 可以把text换成任意字符串
- 甚至可以把emoji加进去
