	- 想用 就必须带上模块的名字
		- my_file.pi
		- 中间`.`的意思是 `的`
		- my_file 模块中的 pi属性

### 名字空间

- 就像 
	- 访问 time模块中的 
	- asctime函数
	- time.asctime()

```
import time
print(time.asctime)
```

- 名字空间 就是 
	- 模块的名字
	- 成员的所有者
- 直接找pi找不到	
	- 带着名字空间 
		- my_file.file
	- 就可以找到了
