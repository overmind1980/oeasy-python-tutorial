def draw_pine(n=5,color="white"):

def draw_pine_head(n,color="white"):

    print("\33["+d_colors[color]+"m",end="")

### 表驱动(table-driven method)

- 方法来自于c语言
	- 用序号来驱动
	- 指向字符串的指针数组的元素位置
- 从而代替if、switch的方法

### 默认细节

```
drawpine()
```

- 这个是全默认
	- n = 5
	- color = "green"

```
drawpine(6)
```

- 这个color是默认
- n = 6
- 如果我想反过来呢？
	- n是默认
	- color不默认

```
drawpine(color="yellow")
```

- 因为第一个参数是n
- 我们想要设置第2个参数color不能直接设置
- 需要用参数的关键字color
- 这样就可以只设置color了
