### 抄写下来，进行分析

```python
import time
i = 1
while i <= 3:
	print(i)
	i = i+1
	time.sleep(1)
```

- 引入了 `time` 包
- i是一个数
	- 一开始等于 `1`
- 注意 `while` 这一行
	- 条件是` i<=3`
	- 行尾有个半角冒号
	- 后面用<kbd>tab</kbd>空了4个字符
	- 在while下紧跟着空了四个字符的三句话
	- 都是while要执行的循环体
	- 注意缩进一定是4个字符
- time.sleep(1)
	- 是time包里面的
	- 小括号应该对应函数
	- sleep(1)这个函数的参数是1
	- 应该是愣1秒的意思

### 进行改造

```python
i = 1
while True:
	print(i)
	i = i+1
	time.sleep(1)
```

- 原来只能循环3次
	- 主要就是while的条件改成了True
	- 注意`True`的大小写
	- 现在无限循环
	- 跟着while但是缩进的三句话无限循环
- `:w|!python3 %`开始循环
	- <kbd>ctrl</kbd>+<kbd>c</kbd>跳出循环回vi
