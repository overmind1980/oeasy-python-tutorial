### 药方输出

```
s_fangzi = "麻黄9g 炙甘草6g 杏仁9g 生石膏15-30g(先煎) 桂枝9g 泽泻9g 猪苓9g 白术9g 茯苓15g 柴胡16g 黄芩6g 姜半夏9g 生姜9g 紫菀9g 冬花9g 射干9g 细辛6g 山药12g 枳实6g 陈皮6g 藿香9g"
l_drugs = s_fangzi.split(" ")
i_num = 0
for s_drug in l_drugs:
    i_screen_len = 0
    for c in s_drug:
        if 0x4e00 <= ord(c) <= 0x9fff:
            i_screen_len += 2
        else:
            i_screen_len += 1
    if i_screen_len >= 8:
        s_drug = s_drug + "\t"
    elif i_screen_len >= 4:
        s_drug = s_drug + "\t\t"
    print(s_drug,end='')
    if i_num % 4 == 3:
        print()
    i_num = i_num + 1
```
