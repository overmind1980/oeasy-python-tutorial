### 多次引用

- 存'oeasy' 的 
	- 这块内存地址
	- 是`唯一`的
- 这块地址 
	- 被 `两个`变量 所引用
		- s1
		- s2
	- 两个变量都引用 这个房间号

![图片描述](https://doc.shiyanlou.com/courses/3584/labs/2686345/uid1190679-20241027-1729996500670) 

- 但是 如果 
	- 又 给s1 再赋个`新`值
	- 那么 s1
		- 会 引用 `新`的地址 吗？🤔

### 实践

```
s1 = "oeasy"
s2 = "oeasy"
print(id(s1))
print(id(s2))
print(id(s1) == id(s2) )
s2 = "o2z"
print(id(s1))
print(id(s2))
print(id(s1) == id(s2) )
```

- s2 最初 引用的是 
	- 存"oeasy"的地址
