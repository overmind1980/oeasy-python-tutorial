# 编码进化

## 回忆上次内容

- 上次 回顾了 计算机存储单位的演变
	- 最小的读写单位 是 bit

- 8-bit 固定下来 成为了字节(Byte) 

| 位数 | 容量 |
|---|---|
| 8-bit | 1Byte |
| 1024Byte	| 1 KB | 
| 1024 KB | 1 MB |
| 1024 MB | 1 GB |
| 1024 GB | 1 TB |

- 存储字符时
	- 第1位 是 标志位
	- 后7位 是 ascii具体的值
		- 可以用 1Byte 存储
- 计算机之间 有了 更多的通信
	- 编码统一 为的是 更好地通信
- 计算机之间 
	- 是`怎么` 开始通信的 呢？？🤔
