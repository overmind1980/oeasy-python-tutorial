### sleep.py

```python
#!/usr/bin/python3
import time
localtime = time.strftime(' %H:%M:%S',time.localtime())
print(localtime,end="")
```

- strftime 函数
  - str 是字符串 string 的意思
  - f 是 format
  - 他把一个 time.localtime() 格式化为一个字符串
- 格式化成什么格式呢？
	- 格式化成 ` %H:%M:%S` 这个格式
	- 这个格式选取了三个时间元组项
		- %H是小时
		- %M是分钟
		- %S是秒钟
		- 中间用:分割
