### 规则原理

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210225-1614230960132)

- 0-7 是 标准色
	- 原来的30-37
- 8-15 是 高亮色
	- 原来的90-97

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210225-1614230607701)

- 16-231 是 混合色
  - r、g、b 来自于
	- 从 0到5 的阶梯范围
    - 具体的颜色数值
	- 来自于 `16 + 36*r + 6*g+ b`
	- 得到数值后 可以显示颜色
- 232-255 是 24级 灰阶色
