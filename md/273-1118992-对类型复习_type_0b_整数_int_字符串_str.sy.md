### 二进制 binary字符串 转 十进制整数

- 用int函数
	- 将二进制的字符串形态
	- 转化为整型数字

```
int("0b101001", base = 2)
int("101001", base = 2)
int("101001", 2)
```

- int的意思是integer
	- 整数

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210817-1629190293619)

- 这三条其实是等价的
  - "0b..."明确是二进制数字
  - base 是参数的名字
	- 标识着 用的是多少进制
