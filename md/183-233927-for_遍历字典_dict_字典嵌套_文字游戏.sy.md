```

for key in l:
	print(key,l[key],sep=":")
```

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20240326-1711421892758)

	- 改变了大小

### 提前定义key

- 提前定义key	
	- 本地变量字典 大小就不变了 

```
key = ""
l = locals()
for key in l:
	print(key,l[key],sep=":")
```

### 总结

- 这次遍历了字典
- 字典在 for 循环中的遍历中
  - 遍历的是字典的 keys()
- 除了 keys()之外，还可以
  - 遍历 values()
  - 遍历 items()
- keys()最简单明确
- items()最全面
- 好像形成 dict 字典的时候可以用一个 zip 函数
