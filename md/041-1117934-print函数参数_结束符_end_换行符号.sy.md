### 条件语句

- 如果 num 能 被8整除
	- 就输出 换行
	- print()

```python

        print()
    print(num, chr(num), sep=":", end="\t")
```

### if语句

- if语句结尾处
	- 要有一个冒号(:)
	- 意味着 下面缩进的 
		- 归if管
		- 都是 if语句的 条件体

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20240508-1715134042011)

- if 的条件是
	- num % 8 == 0
	- 意思是 num 除8余0 时
		- 才执行 if 对应的条件体
- if对应的条件体 是
	- print()
		- 相当于 print(end="\n")
		- 虽然没有输出任何字符
		- 但是有回车换行的效果
- print前面 需要进行更深层的缩进
	- 2个tab
	- 或者 8个空格
		- 才能让print()这句话
		- 归 if语句 管着
