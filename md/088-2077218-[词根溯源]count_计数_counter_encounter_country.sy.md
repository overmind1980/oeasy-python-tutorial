# 列表 - 计数 - count

## 回忆

- 上- 这次研究了计数函数count
	- 可以统计列表中某个列表项出现的次数
	- 也可以统计某字符在字符串中出现的次数
- count与len不同
	- count统计某个列表项的出现次数
	- len统计整个列表的长度
- count 这个单词怎么来的呢？🤔
