### 区别

- 单个列表 看不出 区别
	- 两个列表 就有区别了

```python3

```

- 首先定义 l1、l2是两个列表对象
	- 列表对象l1调用clear函数之后

### 总结一下 新单词

| 分类 | 名称 | 描述 |
| ---- | ---- | ---- |
| `list` 列表 | `append` | 用于在列表的末尾添加一个元素 |
| `list` 列表 | `clear` | 用于清空列表中的所有元素<br/>使列表变为空列表 |
| 编程概念 | `function`（函数） | 一段独立的代码块<br/>能完成特定任务<br/>可直接调用<br/>如 `len()` 函数用于获取列表长度 |
| 编程概念 | `method`（方法） | 与特定对象关联的函数<br/>通过对象来调用，如列表的 `append` 和 `clear` 需通过列表对象调用 |
