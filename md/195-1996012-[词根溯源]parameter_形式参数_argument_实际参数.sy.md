# 函数参数

## 回忆

- 对于程序员来说
	- 函数就是封装好的一个套路
	- 这封装好的套路也可以接收参数
	- 就像洗衣机、电饭煲一样可以设定一些模式
- 传递的参数有两部分两种
	- 函数定义的时候，给出的是一个形式上的样子，是形参(parameter)
	- 函数调用的时候，给出的是具体的实际参数的值，是实参(argument)
	- 调用的时候完成一个赋值的过程
		- 把实参的值赋给形参
- parameter这个单词怎么来的呢？？🤔
